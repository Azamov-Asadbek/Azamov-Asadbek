## About

I build frontend systems that are built to last. Over 5 years I have shipped production interfaces across government platforms, e-commerce ecosystems, and high-traffic media products — taking each from zero to deployed, then owning maintenance over time.

I think in components, architect by responsibility, and write code that teammates can actually work with. My work spans complex admin panels, real-time map dashboards, data-heavy reporting tools, and polished client-facing storefronts.

> Available for remote collaboration. I deliver working products, communicate clearly about tradeoffs, and take ownership from spec to deploy.
